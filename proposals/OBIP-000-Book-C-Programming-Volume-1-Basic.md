## 教材主题

本书主要讲述 C 语言的基础知识，包括一些计算机先修知识，C 语言的基本编程要素和方法，例如：数据类型、运算符、表达式、语句、函数、指针、数组、结构体、共用体、枚举、预处理器（宏）等。

本书的目标是让用户能够在 Unix 或 Windows 下，使用 C 语言编写能进行简单 IO 操作的程序，能够帮助用户理解计算机的基本工作原理。
